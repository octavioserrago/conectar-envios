Alcance del Proyecto

Objetivos del Proyecto

El proyecto Conectar Envíos tiene como objetivo principal facilitar la gestión de envíos a través de una plataforma fácil de usar tanto para clientes como para transportistas. Los usuarios podrán:

  • Registrar sus datos personales y acceder de manera segura a la plataforma.
  • Crear envíos, seleccionar transportistas y realizar un seguimiento en tiempo real de los envíos.
  • Gestionar pedidos y optimizar rutas de transporte.

Alcance Funcional

Este proyecto cubre las siguientes funcionalidades principales:

  • Registro y login de clientes y transportistas.
  • Creación de envíos y seguimiento de su estado.
  • Optimización de rutas para transportistas.
  • Notificaciones sobre el estado de los envíos.

Limitaciones y Exclusiones

Las siguientes áreas están fuera del alcance inicial del proyecto:

  • Soporte para plataformas de pago en línea.
  • Aplicación para dispositivos móviles (solo se contempla la versión web).

Plazo de Implementación

El desarrollo completo de la plataforma Conectar Envíos está planificado para un período de 1 año, incluyendo las siguientes fases:

  • Fase 1: Análisis y diseño - 3 meses.
  • Fase 2: Desarrollo de la aplicación web - 7 meses.
  • Fase 3: Pruebas y despliegue - 2 meses.
